First, clone the repo via git:

```bash

$ git clone https://github.com/zce/chrome-markdown.git -b master
```

And then install dependencies.

```bash
$ cd chrome-markdown
$ npm install
```

#### Build it

```bash
$ gulp
```

*Note: requires a node version >= 4 and an npm version >= 2.*
